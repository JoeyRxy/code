# 事件和概率的关系
1. 互不相容、对立、相等、包含、不可能 等概念都是由事件定义得来的，逻辑是 “事件 推出 概率”，但是概率的结果符合，却未必是这个事件。例如
   1. P(A)=0，A不一定是 “不可能事件”
   2. P(A)=1，A不一定是 “必然事件”

   为什么呢？可能的原因之一是：对于连续型随机变量，有限个点的值发生变化，并不能改变概率结果
2. 独立定义是由概率关系的来的
