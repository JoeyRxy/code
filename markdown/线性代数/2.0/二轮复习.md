## 伴随矩阵
1. 求逆和求伴随可交换 ： $(A^*)^{-1}=(A^{-1})^*$
2. $E^*=E$

## $A^*,A^{-1}$各种性质

| -------- | ------------------------------ | ------------------------------------------------------- | ---------- | ------------ |
| $A^*$    | $(kA)^*=k^{n-1}A^*$            | $\operatorname{det}(A^*)=[\operatorname{det}(A)]^{n-1}$ |
| $A^{-1}$ | $(kA)^{-1}=\dfrac{1}{k}A^{-1}$ | -                                                       |
