# 通过时域乘积将某个系统函数$H_1(z)$编程系统函数$H_2(z)$

需要利用的性质：$h[n]z_0^n\Rightarrow H(\frac{z}{z_0})$

需要注意的事项：除法关系:中是将$H_1(z)$的点$z_1$转换到了$H_2(z)$中的点$z_2=z_1z_0$；而不是$z_1/z_0$
